
# 01 — Introduction: Machine Learning (ML)

## Machine Learning কী?
Machine Learning হলো এমন একটি পদ্ধতি যেখানে কম্পিউটার ডেটা দেখে “নিজে নিজে” শিখতে পারে এবং সিদ্ধান্ত নিতে পারে। অনেক ক্ষেত্রে সব নিয়ম হাতে লিখে প্রোগ্রাম করা সম্ভব হয় না—তখন ML কাজে লাগে।
সহজ ভাষায়:
আগে (Traditional Programming): Rules (নিয়ম) + Data → Output।
Machine Learning: Data + Output (উদাহরণ) → Model (নিয়ম)
এরপর Model নতুন ডেটা দেখে Output দিতে পারে।

## আমরা প্রতিদিন কোথায় ML ব্যবহার করি?
### 1) Web Search (Google/Bing/Baidu)
তুমি যখন সার্চ করো “how do I make a sushi roll”, সার্চ ইঞ্জিন ML দিয়ে ওয়েবপেজগুলোকে র‍্যাংক করে (কোনটা বেশি প্রাসঙ্গিক)।

### 2) Photo Tagging (Instagram/Snapchat)
ছবিতে কে আছে সেটা চিনে বন্ধুর নাম সাজেস্ট করে—এটা Computer Vision/Face Recognition (ML-এর মাধ্যমে)।

### 3) Movie Recommendation
Star Wars দেখার পর “similar movies” সাজেস্ট করা—Recommendation System (ML)।

### 4) Voice to Text / Voice Assistant
Siri/Google কে কিছু বললে কথা বুঝে কাজ করা—Speech Recognition + Language Understanding (ML)।

### 5) Spam Filter
“Congratulations you won a million dollars” টাইপ ইমেইল স্প্যাম হিসেবে ধরা—Classification (ML)।

## Key Terms (Glossary)
- **ML:** ডেটা থেকে শেখা
- **Computer Vision:** ছবি/ভিডিও বোঝা
- **Recommendation:** পছন্দ অনুযায়ী সাজেশন
- **Speech Recognition:** কথা → টেক্সট
- **Spam Filter:** স্প্যাম বনাম আসল মেইল আলাদা করা

## Quick Revision
- ML = ডেটা দেখে শেখা
- সার্চ, ট্যাগিং, রেকমেন্ডেশন, ভয়েস, স্প্যাম—সব জায়গায় ML আছে
