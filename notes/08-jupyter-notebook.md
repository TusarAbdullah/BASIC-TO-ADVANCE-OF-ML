# Jupyter Notebook — Machine Learning শেখার সবচেয়ে জনপ্রিয় টুল

## 1) কেন Jupyter Notebook ব্যবহার করা হয়?
Machine Learning এবং Data Science প্র্যাকটিশনাররা সবচেয়ে বেশি যে টুলটা ব্যবহার করে সেটা হলো **Jupyter Notebook**।  
এটা এমন একটা environment যেখানে তুমি:

- ছোট ছোট অংশে কোড রান করে দেখতে পারো
- experiment করতে পারো (try & test)
- কোড + ব্যাখ্যা একই জায়গায় লিখে রাখতে পারো

এই কোর্সে ওয়েব ব্রাউজারের ভিতরেই তুমি Jupyter Notebook ব্যবহার করতে পারবে।  
এটা কোনো “fake/simplified” টুল না—বাস্তবে বড় কোম্পানিগুলোও একই টুল ব্যবহার করে।

---

## 2) এই কোর্সে কী ধরনের ল্যাব থাকবে?
### 2.1 Optional Labs (খুব সহজ)
Optional lab-গুলো এমনভাবে বানানো:
- সাধারণত **তোমাকে কোড লিখতে হবে না**
- শুধু notebook খুলে **top to bottom** এক লাইন করে রান করবে
- কোডটা আমরা/কোর্স ইতোমধ্যে দিয়ে রাখবে

কেন এগুলো useful?
- ML কোড দেখতে শিখবে
- “কোড কীভাবে কাজ করে” সেটা হাতে-কলমে বুঝবে
- দ্রুত শেষ করা যায়

> Optional lab মানে বাধ্যতামূলক না—ইচ্ছা করলে না করলেও চলবে।  
> কিন্তু করলে কনসেপ্ট বোঝা অনেক সহজ হয়।

### 2.2 Practice Labs (পরের সপ্তাহ থেকে)
এগুলোতে:
- তোমাকে কিছু **কোড নিজে লিখতে হবে**
- ধীরে ধীরে ML implement করার অভ্যাস তৈরি হবে

---

## 3) Notebook-এর ভিতরে “Cells” কী?
Notebook-এ অনেকগুলো block থাকে, এগুলোকে বলা হয় **cell**।  
দুই ধরনের cell সবচেয়ে বেশি দেখা যায়:

### 3.1 Markdown Cell (Text cell)
- এখানে শুধু লেখা/ব্যাখ্যা থাকে
- কোড না, বরং “কোড কী করবে” তা বোঝায়
- চাইলে তুমি text edit করতে পারো

### 3.2 Code Cell (Python code cell)
- এখানে Python কোড থাকে
- এটা run করলে আউটপুট দেখায় (print/result/plot ইত্যাদি)

---

## 4) Cell রান করার নিয়ম (সবচেয়ে গুরুত্বপূর্ণ)
### ✅ Run করার শর্টকাট: `Shift + Enter`
- **Code cell**-এ `Shift + Enter` দিলে কোড রান হবে  
- **Markdown cell**-এ `Shift + Enter` দিলে লেখা সুন্দর formatting এ “render” হবে (formatted text হয়ে যাবে)

---

## 5) Optional Lab কীভাবে করবা? (Step-by-step)
Notebook খুললে এইভাবে করো:

1. উপরে নিচে **scroll করে** notebook টা একটু দেখে নাও  
2. একটি cell select করো  
3. আগে আন্দাজ করো: “কোডটা কী করবে?”  
4. তারপর `Shift + Enter` দিয়ে রান করো  
5. আউটপুট দেখে বুঝে নাও  
6. চাইলে কোড একটু **change** করে আবার run করে দেখো (experiment)

এভাবে একদম **top to bottom** গেলে পুরো notebook শেষ হয়ে যাবে।

---

## 6) কেন Jupyter Notebook শিখলে সুবিধা?
- ML শেখার সময় ছোট ছোট experiment করতে হয়  
- কোড + নোট একসাথে থাকায় revision সহজ  
- পরে নিজের project/assignment/portfolio বানাতে কাজে লাগে  

---

## Recap (১ মিনিটে)
- Jupyter Notebook = ML/Data Science-এর সবচেয়ে জনপ্রিয় coding environment  
- এই কোর্সে browser-এর ভিতরেই notebook ব্যবহার করবে  
- Cell দুই ধরনের: Markdown (text), Code (python)  
- `Shift + Enter` = run/render  
- Optional labs: শুধু run করলেই হবে  
- Practice labs: পরে নিজে কোড লিখবে
